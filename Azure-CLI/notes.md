# Azure CLI Learning Notes

## What is CLI?
A Command Line Interface that allows users to interact with the system through text commands.

## What is Azure CLI?
Azure CLI is a cross-platform command-line tool that is used to manage Azure resources.

## Why Azure CLI?
- Faster than the portal
- Scriptable
- Automatable
- Works across OS (Windows, Mac, Linux)
- Useful for DevOps, IaC, and pipelines
