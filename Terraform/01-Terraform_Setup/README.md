# Terraform Setup and First Deployment

This project demonstrates provisioning an Azure Resource Group using Terraform.

## Tools Used
- Terraform
- Azure CLI
- VS Code

## Steps Performed
1. Installed Terraform
2. Configured Azure Provider
3. Authenticated using Azure CLI
4. Created Terraform configuration
5. Ran terraform init
6. Ran terraform plan
7. Ran terraform apply

## Result
Terraform created a Resource Group in Azure:

Name: terraform-demo-rg  
Region: East US
