## What is Git?
Git is a distributed version control system that tracks changes in source code.

## Core Git Concepts
- Working Directory
- Staging Area
- Local Repository
- Remote Repository (GitHub)

## Git Basic Commands
- git status
- git add
- git commit
- git push
- git pull

## Branching Commands
- git branch <name>
- git checkout <name>
- git merge <branch>

## Why Feature Branches Are Important

- Keep main branch stable and production-ready.
- Allow developers to work independently.
- Prevent unfinished code from affecting others.
- Make collaboration easy through Pull Requests.
- Enable code reviews before merging.
